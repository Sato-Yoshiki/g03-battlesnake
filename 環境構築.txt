・WSLを使う方法
1.wslの起動
wsl
をコマンドプロンプトで打つ

エラーが出たら
wsl -l -v
を打ってみてUbuntuという文字があれば
wsl -s Ubuntu
wsl
を打つ

エラーが出たり、Ubuntuの文字がでなければ
wslのインストールをするために
wsl --install
を打つ
うまく行けばユーザー名とパスワードの設定ができる


2.flaskのインストール
sudo apt install python3-pip
pip install flask

うまく行かなかったら
sudo apt update && sudo apt upgrade
をしてからもう一度

3.ブラウザ(Google Chromeのダウンロード)

1.
cd /tmp
2.
wget https://dl.google.com/linux/direct/google-chrome-stable_current_amd64.deb
3.
sudo dpkg -i google-chrome-stable_current_amd64.deb
4.
sudo apt install --fix-broken -y
5.
sudo dpkg -i google-chrome-stable_current_amd64.deb
うまく行けば
google-chrome
で起動できる
